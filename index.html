지식 증권거래소의 핵심 개념을 알아보세요

✏️
지식 입력 = 주식 발행
  • 섹터(수학·영어·과학 등)를 선택하고 배운 내용을 입력
  • 입력 한 건 = 주식 1주 발행 (주가는 올라가지 않음)
  • 입력할 때마다 C5 보상을 받아요
📈
매수 = 주가 상승
  • 다른 사람의 지식 주식을 C로 매수할 수 있어요
  • 매수할 때마다 해당 주식의 주가가 오릅니다
  • 내 주식도 다른 사람이 매수하면 주가가 오릅니다!
📉
주가 하락
  • 지식을 입력하지 않거나 매수가 뜸해지면 주가가 자동으로 하락해요
  • 꾸준히 지식을 입력하고 관심을 받아야 주가가 유지됩니다
🔄
복습 = 주가 회복
  • 내 주식의 주가가 떨어졌을 때, 복습으로 회복시킬 수 있어요
  • 완벽 기억 +35pt · 흐릿한 기억 +20pt · 기억 안 남 +0pt
💼
포트폴리오
  • 내가 발행한 주식 + 매입한 주식을 한눈에 볼 수 있어요
  • 매입한 주식의 주가가 오르면 평가액도 올라갑니다
